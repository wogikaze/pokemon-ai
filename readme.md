## 起動方法

### 簡単 ver

`$ run`
`$ python eval.py`


### 詳細 ver

Pokemon Showdown のサーバー起動
`$ cd pokemon-showdown ; node pokemon-showdown start --no-security`
python で AI の起動
`$ python eval.py`
